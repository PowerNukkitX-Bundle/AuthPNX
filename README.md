# AuthPNX
A authentication plugin for your (offline mode) PNX server

This plugin is an authentication plugin for your PowerNukkitX server with xboxAuth disabled.

All messages are changable in messages.yml
For storage, you can use h2 (for smaller servers) or mysql.
Passwords are hashed and salted using the PBKDF2 password hashing algorithm.

This plugin also allows you to use microsoft authentication for login.
